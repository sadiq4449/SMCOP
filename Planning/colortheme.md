# 🎨 Portal UI Color Theme Specification

This document defines the color system used in the Portal UI (Government/ERP-style Dashboard).

---

## 🟦 1. Primary Brand Colors

Used for main UI identity, navigation, and primary actions.

- **Primary Blue (Main Brand)**
  - Hex: `#1E3A8A`
  - Usage:
    - Sidebar background
    - Primary buttons
    - Active navigation items
    - Headers / highlights

- **Secondary Blue (Accent)**
  - Hex: `#3B82F6`
  - Usage:
    - Links
    - Hover states
    - Active indicators
    - Info highlights

- **Light Blue (Soft Accent)**
  - Hex: `#60A5FA`
  - Usage:
    - Background highlights
    - Soft UI elements
    - Badge accents

---

## ⚪ 2. Background Colors

Used for layout structure and readability.

- **Main Background**
  - Hex: `#FFFFFF`
  - Usage:
    - Page background
    - Cards
    - Forms

- **Section Background (Light Gray)**
  - Hex: `#F8FAFC`
  - Usage:
    - Panels
    - Dashboard sections

- **Muted Background**
  - Hex: `#F3F4F6`
  - Usage:
    - Tables
    - Alternate rows
    - Sub-sections

---

## ⚫ 3. Text Colors

Used for typography hierarchy.

- **Primary Text**
  - Hex: `#111827`
  - Usage:
    - Headings
    - Important labels

- **Secondary Text**
  - Hex: `#374151`
  - Usage:
    - Paragraphs
    - Supporting text

- **Muted Text**
  - Hex: `#6B7280`
  - Usage:
    - Hints
    - Placeholder text
    - Disabled states

---

## 🟩 4. Status Colors

Used for system feedback and alerts.

- **Success**
  - Hex: `#10B981`
  - Usage:
    - Success messages
    - Approved status

- **Warning**
  - Hex: `#F59E0B`
  - Usage:
    - Alerts
    - Pending status

- **Error / Danger**
  - Hex: `#EF4444`
  - Usage:
    - Errors
    - Rejected status

- **Info**
  - Hex: `#3B82F6`
  - Usage:
    - Information messages

---

## 🧩 5. UI Style Direction

- Design Style: **Government ERP / School Monitoring System**
- Look & Feel: Clean, structured, data-focused
- Framework Inspiration: Tailwind CSS + Modern Admin Dashboards
- Layout: Sidebar-based navigation with card-driven content
- Spacing: Spacious and readable UI
- Shadows: Soft and subtle for depth

---

## 📌 Summary

The system uses a **professional blue-based theme**:

> Deep Blue + White + Light Gray + Sky Blue Accent

This ensures:
- High readability
- Government-grade UI trust
- Modern dashboard feel
