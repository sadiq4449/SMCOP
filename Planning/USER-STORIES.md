School Monitoring & Reporting System — User Stories
1. Super Admin User Stories

(Full System Access)

1.1 System Management
As a Super Admin, I want to create, edit, and delete users so that I can control who accesses the system.
As a Super Admin, I want to assign roles so that each user has proper permissions.
As a Super Admin, I want to manage schools (Add/Edit/Delete) so I can keep school data updated.
As a Super Admin, I want to manage districts, clusters, and classes to control educational structures.
As a Super Admin, I want to configure system settings so the platform behaves as needed.
As a Super Admin, I want to assign schools to Principals and DEOs so their workspace is controlled.
1.2 Monitoring & Reporting
As a Super Admin, I want to view analytics for all districts so I can evaluate system-wide performance.
As a Super Admin, I want to view and override any report to maintain data accuracy.
As a Super Admin, I want to download full datasets to perform external analysis.
1.3 Permissions & Audits
As a Super Admin, I want to set access control rules so unauthorized users cannot edit core data.
As a Super Admin, I want to view audit logs to track suspicious activity.
1.4 Form & Survey Management
As a Super Admin, I want to create/edit custom forms for monitoring.
As a Super Admin, I want to publish forms to DEOs/Enumerators for field visits.
2. Government User Stories

(Monitoring Only – No Edit/Delete Permissions)

2.1 Monitoring
As a Government User, I want to view district dashboards so I can track KPIs.
As a Government User, I want to view school reports to oversee performance.
As a Government User, I want to compare districts to detect strengths/weaknesses.
As a Government User, I want to see school visit data to check inspection progress.
2.2 Review & Feedback
As a Government User, I want to add comments on reports so I can provide feedback.
As a Government User, I want to raise flags or issues for underperforming schools.
2.3 Read-Only Access
As a Government User, I want to view school profiles without editing them.
As a Government User, I want to view timetables, attendance, and teacher profiles but not modify them.
As a Government User, I want to download reports for documentation.
3. DEO (District Education Officer) User Stories
3.1 District-Level Management
As a DEO, I want to view all schools in my district so I can monitor performance.
As a DEO, I want to approve or reject principal reports so data remains accurate.
As a DEO, I want to assign field enumerators for school visits.
3.2 Monitoring
As a DEO, I want to see district-level dashboards to track KPIs.
As a DEO, I want to review enumerator visit reports for quality assurance.
3.3 Task Management
As a DEO, I want to assign improvement tasks to Principals when issues are detected.
4. Principal User Stories
4.1 School Management
As a Principal, I want to manage teacher assignments so classes are properly scheduled.
As a Principal, I want to approve teacher attendance to verify presence.
As a Principal, I want to update school timetable for proper planning.
4.2 Reporting
As a Principal, I want to submit infrastructure reports so the district knows our status.
As a Principal, I want to submit daily school reports for compliance.
As a Principal, I want to view school-level analytics to identify gaps.
4.3 Attendance
As a Principal, I want to monitor student and teacher attendance trends to ensure discipline.
5. Teacher User Stories
5.1 Teaching & Attendance
As a Teacher, I want to mark student attendance to maintain daily records.
As a Teacher, I want to mark my self-attendance for time tracking.
As a Teacher, I want to view my timetable to know my daily schedule.
5.2 Academic Tasks
As a Teacher, I want to upload lesson plans to track curriculum delivery.
As a Teacher, I want to enter student performance/grades for evaluations.
6. Enumerator / Monitoring Officer User Stories
6.1 Visit Management
As an Enumerator, I want to see assigned schools so I know where to visit.
As an Enumerator, I want to submit school visit forms with:
Photos
GPS
Notes
Ratings
As an Enumerator, I want to submit reports offline (optional future feature).
6.2 Evidence Collection
As an Enumerator, I want to upload geo-tagged photos to verify my visit.
7. Shared User Stories
7.1 Authentication
As any user, I want to log in securely to protect my data.
As any user, I want to recover my password in case I forget it.
7.2 Notifications
As any user, I want to receive notifications for tasks, approvals, or alerts.
7.3 Search & Filters
As any user, I want to filter data (schools, teachers, reports) to quickly find information.
8. System-Level User Stories
8.1 Dashboards
As a system user, I want to view charts and analytics to understand performance visually.
8.2 Reports
As a system user, I want to download reports in PDF/Excel format for offline usage.
8.3 Issue Tracking
As a system user, I want to open and track issues like missing facilities or teacher gaps.
